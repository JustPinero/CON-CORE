## Objective

Implement "DEPLOY SCHEDULE" action: select a template + date range, batch-write time blocks as calendar events for each matching day in the range. Weekday templates apply to Mon-Fri, weekend templates to Sat-Sun.

## Acceptance Criteria

- [ ] Select template from dropdown
- [ ] Select date range (start date + end date)
- [ ] Preview shows all events that will be created
- [ ] DEPLOY button batch-creates events via Calendar API
- [ ] Weekday template applies to Mon-Fri dates only
- [ ] Weekend template applies to Sat-Sun dates only
- [ ] Progress feedback during batch creation
- [ ] Success message with event count

## Files to Touch

src/stations/schedule/DeploySchedule.tsx, api/calendar/batch-create.ts

## Tests to Write

- Deploy creates events only for matching day type
- Weekday template skips weekends
- Weekend template skips weekdays
- Batch create sends correct events to Calendar API
- Date range generates correct list of dates

## Dependencies

3.1, 3.2 (Calendar API, templates)
