## Objective

Build schedule template CRUD: create, read, update, delete weekday and weekend templates. Each template has a name, day type, and array of time blocks. Stored in Supabase schedule_templates table. Template editor UI with add/remove time block functionality.

## Acceptance Criteria

- [ ] Create new template with name, day type, time blocks
- [ ] Edit existing template (rename, add/remove time blocks)
- [ ] Delete template with confirmation
- [ ] List all templates
- [ ] Time blocks have: start_time, end_time, label, calendar_id
- [ ] Validation: no overlapping time blocks, end > start
- [ ] Template editor UI with CRT styling

## Files to Touch

src/stations/schedule/ScheduleStation.tsx, src/stations/schedule/TemplateEditor.tsx

## Tests to Write

- Create template stores in Supabase
- Edit template updates correctly
- Delete template removes from Supabase
- Time block validation rejects overlaps
- Time block validation rejects end <= start
- Template editor renders time block list

## Dependencies

3.1, 1.2, 1.4 (Calendar API, CRT design, Shell)
