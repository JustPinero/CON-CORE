## Objective

Register Terminal commands for Schedule and Subscriptions stations. All commands share service functions with GUI.

## Acceptance Criteria

- [ ] `schedule list-templates` outputs template list
- [ ] `schedule deploy <template> <start-date> <end-date>` deploys schedule
- [ ] `schedule conflicts <start-date> <end-date>` checks conflicts
- [ ] `subscr list` outputs subscription list with costs
- [ ] `subscr scan` triggers subscription detection
- [ ] `subscr detail <name>` shows subscription details
- [ ] All commands registered with help text and tab completion

## Files to Touch

src/stations/schedule/schedule.commands.ts, src/stations/subscriptions/subscr.commands.ts, src/stations/terminal/CommandRegistry.ts

## Tests to Write

- All schedule commands call correct service functions
- All subscription commands call correct service functions
- Tab completion works for subcommands
- Commands registered in help system

## Dependencies

3.1-3.7 (all Schedule + Subscription features), 1.8-1.9 (Terminal)
