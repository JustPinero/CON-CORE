## Objective

Scan Gmail for recurring payment receipt patterns using Claude API. Identify subscriptions with: service name, monthly cost, category, first detected date, last charge date. Store in Supabase subscriptions table.

## Acceptance Criteria

- [ ] POST /api/claude/detect-subscriptions scans receipt emails
- [ ] Claude API identifies recurring payment patterns
- [ ] Extracts: service_name, monthly_cost, category, detected_since, last_charge
- [ ] Stores results in Supabase subscriptions table
- [ ] Handles various receipt email formats
- [ ] AbortController timeout on Claude API call
- [ ] Caches results to avoid redundant scans

## Files to Touch

api/claude/detect-subscriptions.ts, src/services/claude.ts

## Tests to Write

- Subscription detection identifies common services
- Cost extraction handles $XX.XX and other formats
- Results cached in Supabase
- AbortController timeout works
- Handles emails with no subscription pattern

## Dependencies

2.1 (Gmail sender data), 1.7 (API structure)
