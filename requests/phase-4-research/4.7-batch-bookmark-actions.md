## Objective

Implement batch actions: delete selected bookmarks, merge duplicates (keep one, delete rest), move selected to different vault, open selected in browser, PURGE DEAD (delete all DEAD bookmarks in vault).

## Acceptance Criteria

- [ ] DELETE SELECTED removes selected bookmarks from Supabase
- [ ] MERGE DUPES keeps highest-quality entry, soft-deletes rest
- [ ] MOVE TO VAULT moves selected to different vault
- [ ] OPEN SELECTED opens URLs in new tabs
- [ ] PURGE DEAD deletes all DEAD bookmarks in current vault
- [ ] Confirmation dialog before destructive actions
- [ ] Action buttons render in ActionButtons component

## Files to Touch

src/stations/research/ResearchStation.tsx, src/services/bookmarks.ts

## Tests to Write

- Delete removes bookmarks (soft delete)
- Merge keeps one, removes rest
- Move updates vault_id
- Purge dead only removes DEAD status bookmarks
- Confirmation required before delete/purge

## Dependencies
