## Objective

Implement dead link detection: send HEAD requests to each bookmark URL, mark as ALIVE or DEAD based on response. "SCAN ALL" trigger scans entire vault. Rate-limited to avoid being blocked.

## Acceptance Criteria

- [ ] HEAD request to each bookmark URL
- [ ] 2xx/3xx = ALIVE, 4xx/5xx/timeout = DEAD
- [ ] Rate limiting: max 5 concurrent requests
- [ ] "SCAN ALL" button triggers full vault scan
- [ ] Progress feedback during scan
- [ ] Status stored in Supabase bookmarks table
- [ ] Handles redirects, timeouts (10s), and network errors

## Files to Touch

src/services/bookmarks.ts (scan logic), src/stations/research/ResearchStation.tsx (scan button)

## Tests to Write

- ALIVE for 200 response
- DEAD for 404 response
- DEAD for timeout
- Rate limiting: max 5 concurrent
- Progress callback fires correctly
- Status written to Supabase

## Dependencies

4.1, 4.4 (bookmark data, UI)
