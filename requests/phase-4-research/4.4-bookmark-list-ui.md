## Objective

Build the bookmark list for a selected vault. Scrollable list with URL, title, status badges (ALIVE/DEAD/DUPE), and dupe count. Selection support for batch actions.

## Acceptance Criteria

- [ ] Bookmark list renders for selected vault
- [ ] Each entry shows: title, URL (truncated), status badge
- [ ] Status badges: ALIVE (green), DEAD (red), DUPE (warning)
- [ ] DUPE badge includes count
- [ ] Multi-select support (checkbox or click-to-toggle)
- [ ] Vault stats at top: total, dead count, dupe count

## Files to Touch

src/stations/research/BookmarkList.tsx, src/stations/research/ResearchStation.tsx

## Tests to Write

- Bookmark list renders bookmarks for vault
- Status badges display correct colors
- DUPE badge shows count
- Multi-select toggles selection
- Vault stats calculate correctly

## Dependencies

4.2, 4.3 (categorized bookmarks, vault nav)
