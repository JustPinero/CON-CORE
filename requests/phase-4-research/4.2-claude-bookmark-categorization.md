## Objective

Send imported bookmarks to Claude API for categorization into vaults (categories). Claude analyzes URLs and titles to assign categories. Results stored in Supabase bookmark_vaults and bookmarks tables.

## Acceptance Criteria

- [ ] POST /api/claude/categorize-bookmarks accepts bookmark list
- [ ] Claude assigns each bookmark to a vault category
- [ ] Vault categories created dynamically based on content
- [ ] Results stored in Supabase (bookmark_vaults + bookmarks)
- [ ] AbortController timeout on Claude API call
- [ ] Batch processing for large bookmark sets (chunk if > 100)

## Files to Touch

api/claude/categorize-bookmarks.ts, src/services/claude.ts, src/services/bookmarks.ts

## Tests to Write

- Categorization returns valid vault assignments
- Large bookmark sets chunked correctly
- Results stored in correct Supabase tables
- AbortController timeout works
- Handles Claude API errors gracefully

## Dependencies

4.1, 1.7 (bookmark import, API structure)
