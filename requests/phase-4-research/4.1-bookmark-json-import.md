## Objective

Implement Chrome bookmarks JSON file upload and parsing. User uploads the Chrome bookmarks export file, parser extracts all bookmarks with URLs and titles from the nested folder structure.

## Acceptance Criteria

- [ ] File upload UI with drag-and-drop support
- [ ] Parse Chrome bookmarks JSON format (nested folders)
- [ ] Extract: url, title, folder path for each bookmark
- [ ] Handle malformed JSON gracefully with error message
- [ ] Display import summary: total bookmarks found, ready for categorization
- [ ] bookmarks.ts service handles parsing logic

## Files to Touch

src/services/bookmarks.ts, src/stations/research/ResearchStation.tsx

## Tests to Write

- Parser extracts bookmarks from nested Chrome JSON
- Parser handles empty bookmark file
- Parser handles malformed JSON
- All URLs and titles extracted correctly
- Folder paths preserved
