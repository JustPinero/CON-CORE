## Objective

Build the horizontal vault category navigation strip. Shows vault names in a scrollable horizontal bar with chunky left/right arrow buttons. Selecting a vault displays its bookmarks below.

## Acceptance Criteria

- [ ] HorizontalNav renders vault categories
- [ ] Left/right arrow buttons scroll categories
- [ ] Selected vault highlighted with inverted colors
- [ ] Clicking a vault triggers selection callback
- [ ] Handles overflow: arrows appear when categories exceed width
- [ ] HorizontalNav is a reusable component

## Files to Touch

src/components/HorizontalNav.tsx, src/stations/research/VaultNav.tsx, src/stations/research/ResearchStation.tsx

## Tests to Write

- HorizontalNav renders all categories
- Arrow buttons scroll the strip
- Selected category has inverted styling
- Click triggers selection callback
- Arrows hidden when all categories fit

## Dependencies

4.2 (vault data), 1.2 (CRT design)
