## Objective

Detect duplicate bookmarks: normalize URLs (strip trailing slashes, www, query params, fragments) and find matches. Mark duplicates with DUPE status and count. Group duplicates for merge.

## Acceptance Criteria

- [ ] URL normalization: strip trailing /, www., query params, fragments, protocol
- [ ] Exact URL match after normalization = duplicate
- [ ] Duplicates marked with DUPE status and count
- [ ] Duplicate groups identified for merge interface
- [ ] Runs automatically after import and on demand

## Files to Touch

src/services/bookmarks.ts (dedup logic)

## Tests to Write

- URLs with trailing slash detected as dupes
- URLs with/without www detected as dupes
- Different protocols (http/https) detected as dupes
- Query params stripped for comparison
- Dupe count accurate
- Non-dupes not falsely flagged

## Dependencies

4.1, 4.2 (bookmark data)
