## Objective

Register Terminal commands for File Recon and Task Queue stations.

## Acceptance Criteria

- [ ] `recon scan` triggers file scan
- [ ] `recon largest` shows largest files
- [ ] `recon oldest` shows oldest files
- [ ] `recon purge` deletes recommended files
- [ ] `queue add <text>` adds to queue
- [ ] `queue next` shows next item
- [ ] `queue list` lists all items
- [ ] `queue pop` removes first item
- [ ] All commands registered with help and tab completion

src/stations/filerecon/filerecon.commands.ts, src/stations/taskqueue/taskqueue.commands.ts, src/stations/terminal/CommandRegistry.ts

## Tests to Write

- Help text registered
- Tab completion works
