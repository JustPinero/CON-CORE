## Objective

Implement the CRT aesthetic system as defined in references/crt-design-system.md. Create global CSS with scanline overlay, phosphor glow, color tokens as CSS custom properties, Courier New typography, border system. Create tokens.ts with color and spacing constants for use in TypeScript. Verify the aesthetic works on a simple test page.

## Acceptance Criteria

- [ ] CSS custom properties defined for all 7 color tokens
- [ ] Scanline overlay renders on all screens via ::after pseudo-element
- [ ] Phosphor text glow via text-shadow on all text elements
- [ ] Courier New / monospace is the only font
- [ ] All borders are 1-2px solid, no border-radius anywhere
- [ ] tokens.ts exports all color and spacing constants
- [ ] No visual regressions when other components are added

## Files to Touch

src/theme/crt.css, src/theme/tokens.ts, src/main.tsx (import crt.css), index.html (font preload)

## Tests to Write

- tokens.ts exports expected color values
- tokens.ts exports expected spacing values
