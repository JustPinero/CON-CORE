## Objective

Add help system and tab completion to Terminal. `help` shows all available commands grouped by station. `help [command]` shows detailed usage. Tab key completes partial commands. Tab with no input shows all commands. Double-tab shows completions when ambiguous.

## Acceptance Criteria

- [ ] `help` lists all registered commands grouped by station
- [ ] `help [command]` shows command usage, description, examples
- [ ] Tab completes partial command names
- [ ] Tab with empty input lists all available commands
- [ ] Ambiguous tab shows possible completions
- [ ] Tab completion works for subcommands (e.g., `comms ` + tab)
- [ ] Help output formatted with CRT styling

## Files to Touch

src/stations/terminal/HelpSystem.ts, src/stations/terminal/TabComplete.ts, src/stations/terminal/TerminalInput.tsx (tab handler), src/stations/terminal/CommandRegistry.ts (metadata for help)

## Tests to Write

- HelpSystem lists all registered commands
- HelpSystem shows detail for specific command
- TabComplete returns single match for unambiguous prefix
- TabComplete returns multiple matches for ambiguous prefix
- TabComplete returns all commands for empty input
- Tab key in input triggers completion

## Dependencies

1.8 (Terminal shell)
