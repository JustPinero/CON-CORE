## Objective

Build the Homepage component: a 3x3 grid of station buttons (COMMS, SCHEDULE, RESEARCH, SECURITY, SUBSCRIPTIONS, CONTACTS, FILE RECON, TASK QUEUE, TERMINAL). Each button shows the station name and F-key shortcut (F1-F9). Clicking a button navigates to that station's route. F-key keyboard shortcuts work globally from the homepage.

## Acceptance Criteria

- [ ] 9 station buttons render in a 3x3 grid
- [ ] Each button shows station name (uppercase) and F-key label
- [ ] Clicking a button navigates to /station/[name]
- [ ] F1-F9 keyboard shortcuts navigate to correct stations
- [ ] Buttons use RetroButton styling (CRT design system)
- [ ] Grid is centered on screen with CRT background
- [ ] "CON-CORE" title/logo area above the grid

## Files to Touch

src/components/Homepage.tsx, src/components/RetroButton.tsx, src/App.tsx (routes)

## Tests to Write

- Homepage renders all 9 station buttons
- Each button has correct station name (uppercase)
- Each button has correct F-key label
- Clicking COMMS button navigates to /station/comms
- F1 key navigates to /station/comms
- F9 key navigates to /station/terminal
- RetroButton renders with correct CRT styling
