## Objective

Build the Shell component that wraps every station: header bar with station name (uppercase) and ESC HOME button, status footer bar. ESC key returns to homepage from any station. All stations render inside this chrome. Create placeholder content for stations that aren't built yet.

## Acceptance Criteria

- [ ] Shell renders header with station name
- [ ] Shell renders ESC HOME button in header (right-aligned)
- [ ] Shell renders status footer bar
- [ ] ESC key navigates to homepage from any station
- [ ] ESC HOME button navigates to homepage
- [ ] Placeholder content shows "STATION OFFLINE" for unbuilt stations
- [ ] Shell uses CRT border and color styling

## Files to Touch

src/components/Shell.tsx, src/components/StatusBar.tsx, src/App.tsx (wrap station routes with Shell)

## Tests to Write

- Shell renders station name in header
- Shell renders ESC HOME button
- ESC key triggers navigation to /
- StatusBar renders with CRT styling
- Placeholder station shows "STATION OFFLINE"
