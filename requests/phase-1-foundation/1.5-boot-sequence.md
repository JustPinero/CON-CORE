## Objective

Build the BootSequence component: a CRT boot animation that plays on initial app load. Sequential line printing with typing effect, shows system initialization messages (loading stations, connecting to APIs, system ready). After completion, transitions to the homepage. Can be skipped with any keypress.

## Acceptance Criteria

- [ ] Boot sequence plays on app load (first visit)
- [ ] Lines appear sequentially with typing effect (50-100ms per character)
- [ ] Messages include: system version, station count, API status
- [ ] Animation completes in ~3-5 seconds
- [ ] Any keypress skips to homepage immediately
- [ ] Boot sequence does not replay on navigation (session flag)
- [ ] Green text on dark background, matches CRT aesthetic

## Files to Touch

src/components/BootSequence.tsx, src/App.tsx (add boot sequence before router)

## Tests to Write

- BootSequence renders initial boot text
- Keypress skips boot sequence
- Boot sequence sets session flag after completion
- Boot sequence does not render if session flag is set

## Dependencies

1.1, 1.2 (CRT aesthetic)
