## Objective

Build the Terminal station: command input with blinking block cursor, output scrollback buffer, command history (up/down arrows), basic command parsing (split by spaces, support quoted strings). Register the Terminal as F9 station. Terminal echoes input and shows "UNKNOWN COMMAND" for unrecognized input.

## Acceptance Criteria

- [ ] Terminal station accessible via F9 or station grid
- [ ] Input field with blinking block cursor (block character)
- [ ] Enter submits command, adds to output buffer
- [ ] Up/down arrows navigate command history
- [ ] Scrollback buffer shows previous commands and output
- [ ] Unrecognized commands show "UNKNOWN COMMAND: [input]"
- [ ] `clear` command clears the scrollback
- [ ] Terminal renders inside Shell chrome

## Files to Touch

src/stations/terminal/TerminalStation.tsx, src/stations/terminal/TerminalInput.tsx, src/stations/terminal/TerminalOutput.tsx, src/stations/terminal/CommandRegistry.ts

## Tests to Write

- TerminalInput renders with blinking cursor
- Enter key submits command text
- Up arrow recalls previous command
- CommandRegistry returns handler for known commands
- CommandRegistry returns null for unknown commands
- Clear command empties output buffer
- Output buffer displays command history
