## Objective

Use Claude API to detect duplicate and near-duplicate contacts. Match by: exact email, similar names (fuzzy match), same phone number. Group duplicates.

## Acceptance Criteria

- [ ] POST /api/claude/detect-contacts-dupes finds duplicates
- [ ] Exact email match = definite duplicate
- [ ] Similar names flagged as potential duplicate
- [ ] Same phone number = likely duplicate
- [ ] Duplicate groups assigned duplicate_group_id
- [ ] Results stored in Supabase

## Files to Touch

api/claude/detect-contacts-dupes.ts, src/services/contacts.ts

## Tests to Write

- Exact email duplicates detected
- Similar name duplicates detected
- Phone number duplicates detected
- Duplicate groups assigned correctly
- Non-duplicates not falsely grouped

## Dependencies

5.4 (contacts data)
