## Objective

Build contact merge interface: show duplicate groups with side-by-side comparison. User selects which fields to keep from each duplicate. Batch merge action combines records.

## Acceptance Criteria

- [ ] Duplicate groups listed with member count
- [ ] Selecting group shows side-by-side comparison
- [ ] User can select preferred field from each duplicate
- [ ] MERGE button combines into single record
- [ ] Batch merge for all groups
- [ ] Merged duplicates soft-deleted
- [ ] CRT styling

## Files to Touch

src/stations/contacts/ContactsStation.tsx, src/stations/contacts/DupeGroups.tsx

## Tests to Write

- Duplicate groups render correctly
- Side-by-side shows all fields
- Merge combines selected fields
- Batch merge processes all groups
- Soft delete on merged records

## Dependencies

5.5, 1.2 (dedup data, CRT design)
