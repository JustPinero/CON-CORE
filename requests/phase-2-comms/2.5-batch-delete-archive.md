## Objective

Implement batch delete and batch archive for emails by sender + category. After selecting a pie chart slice, action buttons appear: DELETE ALL, ARCHIVE ALL, CANCEL. Execute batch operations via Gmail API with rate limiting. Show progress feedback.

## Acceptance Criteria

- [ ] POST /api/gmail/batch-delete accepts sender + category, deletes matching emails
- [ ] POST /api/gmail/batch-archive accepts sender + category, archives matching emails
- [ ] Rate limiting: respects 250 quota units/second with exponential backoff
- [ ] Progress feedback: shows count of processed/total during batch operation
- [ ] Action buttons (DELETE ALL, ARCHIVE ALL, CANCEL) render after slice selection
- [ ] CANCEL button returns to pie chart view
- [ ] Success message shows count of affected emails
- [ ] ActionButtons component is reusable

## Files to Touch

api/gmail/batch-delete.ts, api/gmail/batch-archive.ts, src/stations/comms/SourceIntel.tsx, src/components/ActionButtons.tsx, src/utils/rateLimit.ts

## Tests to Write

- /api/gmail/batch-delete processes emails with rate limiting
- /api/gmail/batch-archive processes emails with rate limiting
- ActionButtons renders correct buttons for comms context
- Rate limiter respects 250 units/second
- Progress callback fires with correct counts
- Cancel button returns to previous view

## Dependencies

2.1, 2.3, 2.4 (sender list, analysis, pie chart)
