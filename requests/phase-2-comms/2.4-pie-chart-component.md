## Objective

Build the PieChart component: SVG donut chart with clickable slices. Green color palette only (using CRT color variations). Clicking a slice selects that category. Shows percentage labels. Used in COMMS to display category breakdown, but built as a reusable component.

## Acceptance Criteria

- [ ] SVG donut chart renders with correct proportions
- [ ] Uses only green palette colors (primary, secondary, tertiary + variations)
- [ ] Each slice is clickable, triggers onSliceClick callback
- [ ] Hover highlights slice (brightness increase)
- [ ] Percentage labels on each slice
- [ ] Handles edge cases: single category (full circle), empty data
- [ ] Accessible: keyboard navigable, aria labels

## Files to Touch

src/components/PieChart.tsx

## Tests to Write

- PieChart renders correct number of slices
- Slice proportions match data percentages
- Click on slice triggers callback with category
- Single-category renders as full circle
- Empty data renders empty state
- All slices use green palette colors
