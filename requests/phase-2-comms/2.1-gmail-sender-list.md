## Objective

Implement Gmail API integration to fetch unique senders with message counts. Server-side route queries Gmail API, aggregates by sender address, returns sorted list. Client-side service calls the route. Handle pagination for large mailboxes.

## Acceptance Criteria

- [ ] GET /api/gmail/senders returns unique senders with message counts
- [ ] Senders sorted by message count (descending)
- [ ] Handles Gmail API pagination (fetch all, not just first page)
- [ ] Respects 250 quota units/second rate limit
- [ ] Returns sender_address, sender_name, message_count per sender
- [ ] Client-side gmail.ts service wraps the API call
- [ ] Error handling for expired tokens (trigger refresh)

## Files to Touch

api/gmail/senders.ts, src/services/gmail.ts

## Tests to Write

- /api/gmail/senders returns expected shape
- Senders sorted by message count
- Rate limiting respects 250 units/second
- Handles pagination correctly
- Handles expired token error

## Dependencies

1.6, 1.7 (auth + API structure)
