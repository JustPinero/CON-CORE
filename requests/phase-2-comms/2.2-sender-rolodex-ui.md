## Objective

Build the COMMS station layout: sender rolodex on the left panel. Scrollable list of senders with message counts, search/filter input at top, chunky UP/DOWN navigation buttons. Selecting a sender highlights it and loads the right panel (placeholder for now). Use the reusable Rolodex component.

## Acceptance Criteria

- [ ] COMMS station renders with left/right panel layout
- [ ] Left panel shows sender rolodex (Rolodex component)
- [ ] Each rolodex entry shows sender name + message count
- [ ] Search input filters senders by name/address
- [ ] UP/DOWN buttons scroll the list
- [ ] Clicking a sender highlights it
- [ ] Selected sender triggers right panel update (placeholder)
- [ ] Rolodex handles empty state ("NO SENDERS FOUND")
